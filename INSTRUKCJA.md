# Instrukcja użycia startera

1. Rozpakuj paczkę `starter_mini_projekt_git_github.zip`.
2. Zmień nazwę folderu na `git-mini-projekt-imie-nazwisko`.
3. W folderze projektu wykonaj `git init`.
4. Wykonaj pierwszy commit ze starterem.
5. Utwórz publiczne repozytorium na GitHub i dodaj je jako `origin`.
6. Utwórz trzy branche opisane w dokumencie Word.
7. Pliki z katalogu `pliki-do-branchy` kopiuj lub włączaj do projektu dopiero na właściwym branchu.
8. Na końcu wypchnij `main` i wszystkie trzy branche na GitHub.

Uwaga: README ma być własne. Starter jest tylko szkieletem.

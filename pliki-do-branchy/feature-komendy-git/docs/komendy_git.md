# Lista komend Git użytych w projekcie

| Komenda | Zastosowanie |
|---|---|
| `git init` | tworzy lokalne repozytorium |
| `git status` | pokazuje stan repozytorium |
| `git add .` | przygotowuje zmiany do commita |
| `git commit -m "opis"` | zapisuje zmianę w historii |
| `git switch -c nazwa` | tworzy i przełącza na nowy branch |
| `git merge nazwa` | scala zmiany z brancha |
| `git remote -v` | pokazuje zdalne repozytoria |
| `git push` | wysyła commity do GitHub |

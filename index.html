Ćwiczenie Git + GitHub

Moja karta pracy Git

Projekt dokumentujący pracę z lokalnym repozytorium, commitami, branchami, merge oraz zdalnym repozytorium GitHub.

Autor: Zuzanna Kania  |  Klasa: 2B

Cel projektu

Projekt powstał w ramach mini projektu ćwiczeniowego z informatyki. Jego celem jest opanowanie podstaw pracy z systemem kontroli wersji Git oraz platformą GitHub: tworzenie commitów, praca na branchach, scalanie zmian i publikowanie kodu publicznie.

  • feature/interfejs-zadania – rozbudowa strony o sekcję commitów, style dodatkowe i test ręczny

Historia commitów

Kluczowe komendy Git

  • git init – inicjalizacja repozytorium
  • git add . – dodanie zmian do staging
  • git commit -m "..." – zapis commita
  • git switch -c feature/nazwa – nowy branch
  • git merge feature/nazwa – scalenie brancha
  • git push origin --all – wypchnięcie wszystkich branchy

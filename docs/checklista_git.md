# Checklista wykonania pracy Git

## Etap 1 – Start projektu
- [x] Rozpakowano pliki startowe
- [x] Zmieniono nazwę folderu na nazwę projektu
- [x] Wykonano `git init`
- [x] Ustawiono gałąź główną: `git branch -M main`
- [x] Dodano pliki: `git add .`
- [x] Wykonano pierwszy commit: „Start projektu ze starterem"
- [x] Utworzono puste publiczne repozytorium na GitHub
- [x] Dodano origin: `git remote add origin <URL>`
- [x] Wypchnięto main: `git push -u origin main`

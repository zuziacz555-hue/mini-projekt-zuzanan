# Opis projektu

## Czym jest ten projekt?

„Moja karta pracy Git" to mini projekt ćwiczeniowy z przedmiotu Informatyka, wykonany w ramach nauki systemu kontroli wersji Git i platformy GitHub. Projekt ma formę prostej strony internetowej z interaktywną checklistą.

## Cel dydaktyczny

Projekt demonstruje umiejętność:
- inicjowania lokalnego repozytorium (`git init`)
- tworzenia commitów z czytelnymi komunikatami
- pracy na oddzielnych gałęziach (branches)
- scalania zmian (`git merge`) do gałęzi `main`
- publikowania repozytorium na GitHub i pracy ze zdalnym origin

## Technologie

- **HTML5** – struktura strony
- **CSS3** – stylowanie i wygląd
- **JavaScript** – licznik zaznaczonych zadań
- **Git** – kontrola wersji
- **GitHub** – zdalne repozytorium

## Uwagi

Projekt jest wynikiem samodzielnej pracy i zawiera oryginalną dokumentację. Historia commitów obrazuje kolejne etapy realizacji zadania.
